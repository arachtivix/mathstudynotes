\section*{Solution}

Here is some example Clojure code:

\begin{lstlisting}
(defn factorial [n]
  (loop [i n acc 1]
    (if (zero? i)
      acc
      (recur (dec i) (* acc i)))))
      
(println (factorial 5)) ; Output: 120
\end{lstlisting}
